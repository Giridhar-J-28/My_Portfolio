## Installation

Clone the repository:

```bash
git clone https://github.com/Giridhar-J-28/My_Portfolio.git
```

Navigate to the project directory:

```bash
cd My_Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Deployment

This website is deployed on Vercel and connected to GitHub for continuous deployment. Any changes pushed to the main branch are automatically deployed.
